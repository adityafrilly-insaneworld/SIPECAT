# Sistem Pencatat Surat Desa (Bukan Simulasi)

## Jalankan lokal
1) Masuk folder `server`
2) `npm install`
3) Copy `.env.example` jadi `.env` lalu ubah `JWT_SECRET` & password admin
4) `npm start`
5) Buka: http://localhost:8080

Login awal (sesuai .env):
- username: SEED_ADMIN_USERNAME
- password: SEED_ADMIN_PASSWORD
